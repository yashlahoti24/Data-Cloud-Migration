Hello, world!

This was a Simple web based cloud application for transmitting the data securely through cloud computing ! This process was completely handled by data Encryption and Decryption handling techniques.

Being a nascent field, it is crowded with many different problems that the engineers and scientist are working assiduously to eliminate. One of the main drawbacks with cloud is security.So, this project proposes a mechanism for secure file storage cloud using encryption and Diffie Hellman. The algorithm involves encrypting the file stored on the cloud and authenticating the user to decrypt the required file.

Upload Files

The first step for the user was to register on our platform . On registering the user would be given an automatically generated private key that would be used by the user for transactions. For security purposes the private key of the user is not stored in the database.

File Directory

Clicking the file directory option on the page takes the user to the above page which displays all the different files stored on the cloud.Selecting different options like download-public key lists the public keys of the registered users which the user can download and use for authenticating the user when someone tries to open his uploaded file.

Register User

Clicking on register user tab take the user to a page where the user registers himself with the platform and generated he Secret Key ( Private ) and Public Keys for the Users. Those public key and private keys cannot be altered and is stored in database. These can be used for file sharing and for decrypting the files which was actually sent by the sender.
